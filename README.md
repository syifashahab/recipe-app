# Recipe App (Fullstack CRUD)

Simple fullstack app untuk manajemen resep menggunakan React, Node.js, dan MongoDB.

---

## Features
- CRUD Recipe
- Pagination
- Filter by level
- Modal add/edit
- MongoDB integration

---

## Tech Stack
- React (Vite)
- Node.js + Express
- MongoDB

---

## Run Frontend
```bash
cd resep-app
npm install
npm run dev
